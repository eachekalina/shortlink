# 002 Применение Redis в качестве кеша

## Дата

2024-06-14

## Контекст

При большом количестве запросов база данных может вызывать слишком большую задержку.

## Решение

Использовать кеш на основе Redis.

## Последствия

Преимущества:
- Уменьшение задержек

Недостатки:
- Необходимость поддерживать Redis
